Rogudator's Comic Panel Generator(ALPHA)

IN ORDER TO USE THIS PLUGIN YOU NEED TO RUN KRITA AS AN ADMINSTRATOR

In order to use this plugin, look at your comic page. How much do you see unique panels vertically? This is the number of rows you need to put in row's section. How much do you see horizontally? This number you put in columns' section. Plugin is in alpha. More functionality will be added later.

Number of rows

This is how much panels will be horizontally.

Slider on the right can change value from 1 to 5. If you need more, you can change value manually by clicking on the number

Number of columns

This is how much panels will be vertically.

Slider on the right can change value from 1 to 5. If you need more, you can change value manually by clicking on the number

Gutter

Gutter is the space betwean panels.

Slider on the right can change value from 1 to 500. If you need more, you can change value manually by clicking on the number

Line thickness

Slider on the right can change value from 1 to 30. If you need more, you can change value manually by clicking on the number
